# PlateKeyPointRegression
车牌关键点定位，全卷积网络预测车牌16个关键点位置。

这里提供大约700+张车牌数据用于训练，包含训练集693，验证集41张。每张图片标注了16个关键点坐标。

具体实现见notebook。
